# Lista profesional de imágenes requeridas — Latam Expeditions

## Regla de carpetas

Todas las imágenes deben subirse en una sola carpeta:

`assets/img/`

No crear subcarpetas por país, tour o paquete. Para mantener orden, usar nombres descriptivos con país + destino + sección + numeración.

Formato recomendado:

`pais-destino-seccion-01.jpg`

Ejemplo:

`peru-machu-picchu-experiencia-01.jpg`

Medidas recomendadas:

- Hero principal: 1920 x 1080 px o superior.
- Cards de experiencias, paquetes y destinos: mínimo 1200 x 900 px.
- Galerías internas de experiencias: mínimo 1400 x 900 px.
- Formato sugerido: `.jpg` optimizado en calidad 75–85%.
- Peso ideal por imagen: entre 250 KB y 600 KB.
